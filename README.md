[![Updated Badge](https://img.shields.io/github/last-commit/kaminskia1/excel-autograder)](https://github.com/kaminskia1/excel-autograder/)

## Abstract
Lightweight auto-grader for Excel-based classroom assignments.
A live demo can be found [here](https://excel.kaminski.pw/).

## Features

1. Automatically grade Excel submissions using Python.
2. Frontend UI built with `Angular` and `TypeScript`.
3. CLI utilities and npm / pip workflows for managing users.
4. Test suites for both Python and JavaScript components.

## Tech stack

1. Python (persistent content logic, Authentication)
2. TypeScript (frontend, grading logic)
3. npm (frontend tooling)
4. pip (Python dependencies)
5. Angular (UI)
